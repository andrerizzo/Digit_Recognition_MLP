# Estudo sobre Redes Neurais Artificiais

Apresento aqui um estudo detalhado sobre Redes Neurais Artificiais, especificamente focado em Perceptrons Multicamadas (MLP) aplicadas ao problema de reconhecimento de dígitos manuscritos utilizando o conjunto de dados MNIST.  
Neste estudo abordo alguns pontos importantes no processo de modelagem tais como:  
* A importância da normalização de dados em ANNs
* Visão geral da arquitetura MLP
* Definição do modelo via Tensorflow
* Funções de ativação
* Hiperparâmetros e otimização
* Desempenho do modelo

Conclusão:  
Pude observar que as ANNs com base em MLPs apresentam acurácia elevada na identificação de dígitos manuscritos em tons de cinza. Tendo em vista a sua relativa simplicidade e baixo custo computacional, constituem uma ótima opção para este cenário de uso.
